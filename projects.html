Funko Collection

EXPRESS | CSS| BOOTSTRAP | EJS | MONGOOSE | MONGODB

Funko Collection is a an app to store all of your favorite Funko Pops. It allows you to add new pops to not your collection. You can also create a wishlist to keep track of all the pops you are looking to buy!
  • This site was built with HTML, CSS.
  • The app is built using express, ejs node.js as well as mongodb and Mongooese for the database.
  • You have create, edit, show, and delete routes for both the collection and wishlist.
  • There is a sort button in the header that allows you to change how you view your collection.
  • It was deisgned using Bootstrap as a front-end FrameWork
  • This site is responsive and looks great from a mobile device.
  • This app is deployed using Heroku
GO TO FUNKO COLLECT

Paws And Found

HTML | CSS | REACT | RUBY ON RAILS | PSQL

Paws and Found is like your neighborhood telephone poll. Its designed to help people post all about their lost pet. You can post a picture, the pets name, the species, breed, last known location, and a description. When the pet is finally found we move that pet on to the found list.
  • Paws and found was built using React on the front-end and ruby on rails for the back-end.
  • Using psql statments to build a table and create the database for the app.
  • The design is soley CSS no framework
  • This app was also a group project me and my classmaste Jake Rick designed
  • On the about page you get to learn a little something about me and jake.
  • This site is responsive and looks great from a mobile device.
  • This app is deployed using Heroku
