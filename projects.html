Funko Collection

EXPRESS | CSS| BOOTSTRAP | EJS | MONGOOSE | MONGODB

Funko Collection is an app to store all of your favorite Funko Pops. It allows you to add new pops to your collection and create a wishlist to track all the pops you are looking to buy!

GO TO FUNKO COLLECT
  • This site was built with HTML, CSS.

  • The app is built using express, ejs node.js, as well as, MONGODB and Mongooese for the database.

  • You have create, edit, show, and delete routes for both the collection and wishlist.

  • There is a sort button in the header that allows you to change how you view your collection.

  • It was designed using Bootstrap as a CSS FrameWork

  • This site is responsive and looks great from a mobile device.

  • This app is deployed using Heroku

Remember It

HTML | ANGULAR.JS | MONGODB | NODE.JS

Remember It is an app deisgned to store the users favorite memories. Whether it is a day at the beach or your wedding day you can store 2 photos, the name of the event, date, location, and a description.

GO TO REMEMBER IT
  • I used bulma as a CSS framework to design this site.

  • The app is built using Angular.js for the front-end and MONGODB with Node.js for the backend database.

  • You have create, update, delete routes for your memories.

  • This site is responsive and looks great from a mobile device.

  • This app is deployed using Heroku.

Paws And Found

HTML | CSS | REACT | RUBY ON RAILS | PSQL

Paws and Found is like your neighborhood telephone poll. You can post the pets name, picture, species, breed, last known location, and a description of your missing pet. When the pet is found we move it onto the found list.

GO TO PAWS AND FOUND
  • Paws and found was built using React on the front-end and ruby on rails for the back-end.

  • Using psql statments to build a table and create the database for the app.

  • The design is soley CSS no framework.

  • This app was a group project. My classmate Jake Rick and I designed it.

  • On the about page you get to learn a little something about jake and I.

  • This site is responsive and looks great from a mobile device.

  • This app is deployed using Heroku.
